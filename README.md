Nova IDE
========

This project is running on Vercel. You can visit it at https://nova-ide-nine.vercel.app/home.html.


Nova IDE is an Web Application designed to work as an IDE for coding Python or creating web pages using HTML, CSS and JavaScript.
The Python Compiler is connected to a third party app called Onecompiler. Onecompiler supports 100+ languages and is used by one of the most large companies and schools. Since we do not want many people to use the interpreters (due to low credits), we have added an early access form at the login page so if you have a proper use for this product, or just want to test it out, you can fill in the form and we will contact you. 
For the AI, I used Google AI studio free tier. I gave it access to the IDE so it can read your changes as it goes by.
